# Bonsai Digital

A single-page portfolio/agency website for Bonsai Digital, a Japan-based digital agency. Built with Next.js, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

```sh
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — Start Next.js development server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Run ESLint
- `npm run test` — Run tests (Jest)
- `npm run test:watch` — Run tests in watch mode

## Tech Stack

- Next.js (App Router)
- TypeScript
- React 18
- Tailwind CSS
- shadcn/ui
- Framer Motion
